# ABB-Simulator-for-Controller-Application-and-HMI-Graphics-Testing
To design and implement a simulator that replicates the behaviour of a controller application and enables the functional testing of corresponding HMI graphics — all within a software-only environment that requires no physical hardware and communicates over a standardised industrial protocol.

This project runs on <<Python 3.13.13>>, on <<Ubuntu WSL2 for Windows 11>>.

Ensure that the below command is run on your system before running the code.

<<pip install pymodbus==3.6.8 streamlit plotly --break-system-packages>>

It installs the necessary dependencies for this project.
